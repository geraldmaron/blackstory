# Asset usage

- Use `black-book-primary-light.svg` on Archive Paper, white, and other approved light surfaces.
- Use `black-book-primary-dark.svg` on Black Ink and approved dark surfaces.
- Use the compact mark at 20-31 CSS pixels.
- Use the primary mark at 32 CSS pixels and above.
- Use the full signature at 168 CSS pixels and above.
- Never type an additional B before the signature. The symbol is the first B in Black Book.
- Use SVG for web and product UI. Use transparent PNG only when the destination cannot render SVG.
