concepts

Signature, palette, type, and map dignity rules for anything users see.

Brand language

Product name: BlackStory.
Core line: History, pinned to place.
Support: People. Places. Evidence. Context.

Copy is specific over sweeping, evidence before assertion, pride without spectacle. Invite; do not lecture. Never sensational framings or completeness overclaims.

Color

Black and paper lead; copper points.

Role Hex
Black Ink #0A0A0A
Charcoal #161616

Copper is a navigational signal (~10–15% of a composition): active locations, selected filters, primary actions, evidence markers. Dark theme is first-class.

Flat matte fills only — no bevels, shadows, glows, or gradients.

Type

  • Sora SemiBold — headlines
  • Inter — UI / body
  • Source Serif 4 — editorial / longform
  • IBM Plex Mono — data, citations, dates, confidence

The mark

Official lockup and book-and-pin symbol are artwork. Never reconstruct either by typing the wordmark beside a bare symbol render.

Map dignity

No red or alarm hues for violence-adjacent records; no crime-heat rendering; color is never the only signal. Points render no sharper than stored precision.
