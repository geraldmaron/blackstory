Install and bootstrap

BlackStory is a TypeScript and Python monorepo. Local development does not require cloud credentials. If you want the product story first, start with Why BlackStory.

Prerequisites

  • Node.js 22+ (nvm use from .nvmrc)
  • pnpm 9.x
  • uv (Python 3.12+)
  • Docker optional for the parked local PostGIS under infra/database/

Bootstrap

git clone https://github.com/geraldmaron/blackstory.git
cd blackstory
./scripts/bootstrap.sh
# or
pnpm bootstrap

Copy .env.example for local emulator-oriented placeholders. Do not put production secrets in the working tree.

Validate

pnpm validate
pnpm test:preflight
pnpm test

Build and typecheck (APIs consume built package declarations):

pnpm build && pnpm typecheck

Run the public web app

pnpm dev:web
# http://localhost:3048/
# http://localhost:3048/explore

The preferred launcher keeps PUBLIC_DATA_SOURCE and DATABASE_URL coherent. Without a Postgres URL you get the small Dunbar seed catalog. With Postgres configured you see the live released catalog.

Firebase emulators (optional)

pnpm firebase:emulators
pnpm firebase:test:rules

This docs site

pnpm --filter @repo/docs dev
# http://localhost:3050/

Production-shaped static export for GitHub Pages:

DOCS_BASE_PATH=/blackstory pnpm --filter @repo/docs build
# or from the monorepo root:
pnpm docs:publish
