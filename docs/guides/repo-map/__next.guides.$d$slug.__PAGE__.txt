<h2 id="stable-code-prefixes"><a class="heading-anchor" href="#stable-code-prefixes">Stable code prefixes</a></h2>
<p>Package scope is brand-agnostic so a product rename does not rewrite the
monorepo:</p>
<table>
<thead>
<tr>
<th>Layer</th>
<th>Stable value</th>
</tr>
</thead>
<tbody>
<tr>
<td>npm packages</td>
<td><code>@repo/*</code></td>
</tr>
<tr>
<td>CSS / tokens</td>
<td><code>ds-*</code> / <code>--ds-*</code></td>
</tr>
<tr>
<td>Env break-glass</td>
<td><code>APP_*</code></td>
</tr>
</tbody>
</table>
<p>User-visible strings and assets come from identity helpers and
<code>apps/web/public/brand/</code> role-based paths.</p>
