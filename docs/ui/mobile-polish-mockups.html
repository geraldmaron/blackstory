Mobile polish · design options

Three modern, compact approaches

Flat matte, Archive Paper, copper as signal only. Type stays medium-weight and small (mastheads 16–17, entity 17–18, rows 13). Differences are surface nesting and how much edition chrome stays on browse tabs.

Ledger Line

Dense index first. Hierarchy from type and hairlines, not stacked cards.

  • Masthead16 Inter
  • Entity17 Inter
  • Section10 Mono
  • Row13 Inter
  • Caption11 Inter
  • Fix: kill card-in-card on History; drop indexed panels on browse tabs; outline count chip on map.
  • Keep: kickers, LedgerRow density, entity fact strips, copper active tab.
  • Best if you want the fastest cleanup and maximum content above the fold.
  • Tradeoff: less “edition object” feel on tabs; closer to a native list app.
